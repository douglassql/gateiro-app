# Domain Overview

Este diretório contém a definição conceitual do domínio do Gateiro App.

## Objetivo

- Modelar entidades principais
- Definir regras de negócio
- Mapear comportamentos da aplicação
- Servir como base para backend e mobile

## Descrição do Domínio

O domínio representa o universo dos cuidados com gatos, incluindo:

- Gestão de pets
- Controle de saúde (vacinas e medicamentos)
- Controle de alimentação e estoque de ração
- Sistema de lembretes
- Conteúdos educativos e dicas para gateiros

Este material é independente de tecnologia e serve como base para decisões arquiteturais futuras.
