# Glossary

## Gateiro
Pessoa responsável pelo cuidado de gatos.

## Pet
Animal cadastrado na aplicação.

## Dose
Quantidade de medicamento administrado.

## Intervalo
Tempo entre administrações de medicamento.

## Estoque
Quantidade disponível de ração.

## Lembrete
Notificação programada para alguma ação.

## Vacinação Preventiva
Vacinação realizada para prevenção de doenças.

## Tratamento Contínuo
Uso prolongado de medicamentos.
