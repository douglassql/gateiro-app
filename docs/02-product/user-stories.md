# User Stories

## Pets
- Como gateiro quero cadastrar meus gatos para organizar informações individuais.

## Vacinas
- Como gateiro quero registrar vacinas para acompanhar o histórico de imunização.

## Medicamentos
- Como gateiro quero registrar tratamentos para lembrar quando medicar.

## Alimentação
- Como gateiro quero controlar ração para saber quando comprar novamente.

## Lembretes
- Como gateiro quero receber notificações para não esquecer cuidados importantes.
