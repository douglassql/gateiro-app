# Backend Architecture

## Estilo Arquitetural

- API REST
- arquitetura em camadas
- separação por domínio

## Camadas

- presentation (controllers)
- application (use cases)
- domain (regras e entidades)
- infrastructure (db, serviços externos)

## Responsabilidades

- validação
- regras de negócio
- persistência
- autenticacao (futuro)
