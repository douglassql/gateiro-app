# System Overview

## Visão Geral

O Gateiro App é uma aplicação mobile focada em organização e cuidados com gatos.

## Componentes Principais

- Aplicação Mobile (React Native)
- API Backend
- Banco de Dados
- Sistema de Notificações

## Responsabilidades

### Mobile
- Interface do usuário
- notificações locais
- gerenciamento de estado

### Backend
- regras de negócio
- persistência de dados
- autenticação
- processamento de lembretes

### Banco
- armazenamento persistente
