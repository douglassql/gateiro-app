# UX Documentation

Documentação de experiência do usuário do Gateiro App.

## Objetivos

- facilitar organização dos cuidados com gatos
- reduzir esforço para registrar informações
- oferecer navegação simples
- permitir uso rápido no dia a dia

## Princípios

- poucos cliques
- interface limpa
- feedback visual claro
- foco em ações rápidas
