# ADR-001 - Estratégia Mobile Local-First

## Status

Aceito

## Contexto

Aplicação focada em execução offline e uso pessoal
durante a fase inicial do projeto.

## Decisão

A aplicação funcionará inicialmente de forma local-first,
sem dependência de backend.

## Consequências

+ desenvolvimento mais rápido
+ testes diretos via APK
+ funcionamento offline
- sincronização futura exigirá planejamento
